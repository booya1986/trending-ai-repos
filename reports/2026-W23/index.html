מה זה עושה

agentsync is a zero-dependency Node.js CLI that converts and synchronises AI coding-agent rule files across formats: AGENTS.md, CLAUDE.md, .cursorrules, GitHub Copilot instructions, Windsurf, Cline, Aider, and Gemini. It can initialise a project by scanning the repo and generating AGENTS.md, convert any rule file to another format, sync one source file to multiple targets, run in watch mode for live regeneration, and install a pre-commit hook to keep files in sync on every commit. A live browser playground is available for trying conversions without installing anything.

למה זה בטרנד

Developers who use multiple AI coding assistants (Claude Code, Cursor, Codex, Copilot) face rule-file drift: the same instructions live in different files in slightly different formats and fall out of sync. agentsync solves this with a single command and no runtime dependencies, making it easy to adopt in any project. The --auto flag can detect which file was most recently edited and propagate changes from it.

דוגמת שימוש

A team using Claude Code internally and Cursor individually runs agentsync setup to create AGENTS.md from a project scan, configure CLAUDE.md and .cursorrules as targets, and install a pre-commit hook. From that point, editing AGENTS.md and committing automatically regenerates all other rule files before the commit lands.

למה זה רלוונטי לך

Directly useful for anyone who works with Claude Code alongside other AI tools: one edit to AGENTS.md keeps all agents aligned without manual copying.

מה זה עושה

sandboxed is the open-source engine for building AI app-builder products. One HTTP POST creates an isolated Linux container for a user; a second POST runs an AI coding agent (Claude Code or OpenCode, pre-installed) inside it with a prompt; the app becomes live at a unique preview URL immediately. Containers sleep when idle (releasing memory) and wake on first request (files persisted on disk), so a single server supports many users. The control plane is a single Go binary using Docker, Traefik for routing and TLS, and SQLite as the database. No Kubernetes, no separate database server, no message queue.

למה זה בטרנד

Products like Lovable, Bolt, v0, and Replit have proven that users want to generate running apps from prompts. sandboxed is the first well-documented open-source backend that replicates this infrastructure in one command on a self-hosted server. At 445 stars within days of launch, it reflects strong pent-up demand for a self-hosted alternative.

דוגמת שימוש

A startup building an AI app-builder deploys sandboxed on a single VPS. When a user submits a prompt, their backend calls POST /sandbox to get an isolated container, then POST .../tasks to send the prompt to Claude Code inside it. Seconds later, the generated app is live at a unique preview URL they show the user.

למה זה רלוונטי לך

If you are building or evaluating an AI-powered development platform, sandboxed is the clearest open-source reference for the multi-tenant sandbox infrastructure layer.

מה זה עושה

Vigils is a local-first control plane (Rust + Tauri 2 desktop app + Chrome MV3 extension) that sits between AI agents and the tools and data they access. It provides four guarantees: a tamper-evident SHA-256 hash-chained audit ledger with full-text search; a default-deny firewall with a Rust policy DSL; a human-in-the-loop approval queue that pauses risky actions for review; and a secret/PII redaction engine that strips 13+ credential classes plus PII before text reaches a model, a log, or the UI. A credential lease broker injects short-lived secrets only into the child process that needs them. The Chrome extension redacts credentials and PII before paste or submit on AI sites including Claude.ai.

למה זה בטרנד

As AI agents take on more file system, network, and API access, the gap between what agents can do and what users can observe has become a real risk. Vigils is one of the first tools to address this comprehensively with local-first guarantees (nothing leaves the machine), a tamper-evident ledger, and both desktop and browser protection.

דוגמת שימוש

A developer uses Vigils alongside Claude Code in a codebase containing API keys. When the agent attempts a file write outside an approved path, the approval queue pauses it for review. Any API keys that appear in prompts are redacted before reaching the model. The audit ledger records every tool call with its hash chain, so the developer can replay the session later to verify what happened.

למה זה רלוונטי לך

A practical local-first safety layer for daily Claude Code work, especially in codebases with credentials. The Chrome extension alone (redacting secrets before pasting into Claude.ai) has immediate value.
