השבוע יש לנו 10 רפוזיטוריות שרלוונטיות במיוחד לתחומים שלך.

מספר 1. lineage skill. Lineage Skill היא כלי Python שמזקק סרטונים, קובצי PDF, תמלילים וסיכומים לכישורי Agent מבוססי מקור. הכלי מאפשר לך להפוך תוכן קיים, כמו הרצאות מוקלטות ותיעוד טכני, לידע מובנה שסוכני AI יכולים לגשת אליו ישירות.

מספר 2. memory os. Memory-OS הוא מערכת הפעלה לזיכרון AI עם 7 שכבות, המספקת זיכרון מתמשך באמצעות Qdrant, הזרקת הקשר מדויקת, ווויקי שמנהל את עצמו. הכלי רץ מקומית ותומך בכל ספק LLM, ומאפשר לסוכני AI לצבור ידע על הפרויקטים שלך לאורך זמן.

מספר 3. munder difflin. Munder Difflin הוא מסגרת TypeScript מקומית להרצת מספר סוכני Claude Code במקביל, עם זיכרון משותף ביניהם. הכלי מאפשר לך לתאם זרימות עבודה מורכבות של סוכנים ללא עלויות ענן נוספות.

מספר 4. skills. LLMQuant Skills היא אוסף כישורים מוכנים לשימוש עבור Claude Code, Cursor ו-Codex, המכוסים בנתוני מימון כמותי מ-LLMQuant. הכישורים מכסים מחקר מניות, מסחר באופציות, ניהול תיקים וניהול סיכונים, ומאפשרים הוספת יכולות ניתוח פיננסי לסוכני AI קיימים.

מספר 5. cc fleet. CC-Fleet הוא כלי Go שמאפשר לך לשלב מודלי LLM של צד שלישי כמו DeepSeek, Qwen ו-MiniMax ישירות כ-Teammates ב-Claude Code. כך תוכל לנתב משימות לפי עלות ויכולת ולהפחית עלויות API תוך שמירה על זרימת העבודה הרגילה שלך.

מספר 6. ultracode skill. Ultracode Skill הוא כישור Codex שמאפשר יצירת תהליכי עבודה קוד דינמיים ומסתגלים. הכישור מפחית את עומס ניהול הצעדים הידניים בעת עבודה עם Codex על משימות קוד מורכבות.

מספר 7. echocut. Echocut הוא כלי CLI מקומי שמשנה צילומי וידאו גולמיים לתוצרי וידאו מוכנים לפרסום עם פקודה אחת, תוך שימוש ב-FFmpeg, WhisperX ו-Ollama. כל העיבוד נעשה מקומית ללא העלאה לענן, מה שמשמר פרטיות ומחסל עלויות API.

מספר 8. paperjury. Paperjury הוא כישור Claude Code שמדמה ביקורת עמיתים על מאמרים מחקריים לפני הגשה, כולל מחזור של סקירה, פסיקה, עדכון ואימות. הוא תומך ב-LaTeX ועוזר לאתר חולשות מתודולוגיות ופערים בניתוח לפני שהמאמר מגיע לסוקרים אמיתיים.

מספר 9. Codex Workflows. Codex-Workflows היא ספריית TypeScript שמאפשרת הגדרת תהליכי עבודה של סוכני AI כקוד, עם תיאום רב-סוכנים דרך MCP. הגישה של workflow-as-code מאפשרת גרסאות, בדיקות ושיתוף של צינורות AI מורכבים כמו כל קוד תוכנה רגיל.

מספר 10. DeepFake AI RealTime. DeepFake-AI-RealTime הוא ערכת כלים C# מבוססת LLM לזיהוי וניתוח תוכן סינתטי בזמן אמת, כולל זיהוי דיפ-פייק בווידאו, אודיו ותמונות. הכלי מתאים לצוותים הבונים מערכות אימות מדיה, ניהול תוכן ובטיחות פלטפורמות.
