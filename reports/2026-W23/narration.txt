השבוע יש לנו 10 רפוזיטוריות שרלוונטיות במיוחד לתחומים שלך.

מספר 1. lineage skill. lineage-skill הוא כלי פייתון שממיר סרטונים, קבצי PDF ותמלולים לכישורי סוכן מגובים במקורות. אם אתה בונה סוכני AI, הוא יכול להפוך את התוכן החינוכי הקיים שלך לכישורים פעילים ישירות.

מספר 2. memory os. memory-os היא מערכת הפעלת זיכרון בת 7 שכבות לסוכן Hermes, עם זיכרון מתמשך ב-Qdrant, ויקי אוטומטי והזרקת הקשר מדויקת, שפועלת מקומית עם כל ספק LLM. עם 892 כוכבים תוך ימים בודדים, זהו אחד הפרויקטים המרגשים ביותר השבוע לכל מי שבונה סוכני AI.

מספר 3. munder difflin. munder-difflin היא רתמת TypeScript בחינם להרצת מספר סוכני Claude Code מקומית עם זיכרון משותף. אם אתה מפתח עם Claude Code ומחפש לתאם מספר סוכנים, זו נקודת התחלה פשוטה ועצמאית.

מספר 4. skills. LLMQuant skills הוא אוסף כישורים לשימוש חוזר לסוכני AI שמכסים מחקר מניות, ETF, אופציות וניהול תיקים, ומגובים בנתוני LLMQuant. אם אתה עובד בפיננסים ומשתמש ב-Claude Code או Cursor, הכישורים האלה מביאים ניתוח פיננסי ישירות לזרימת העבודה שלך.

מספר 5. cc fleet. cc-fleet הוא פרוקסי Go שמאפשר ל-Claude Code לעבוד עם כל ספק LLM חיצוני כגון DeepSeek, Qwen או Kimi כסוכני צוות מקוריים. אם אתה משתמש ב-Claude Code ורוצה לנתב משימות ספציפיות למודלים זולים יותר, כלי זה חוסך החלפת סביבות עבודה.

מספר 6. ultracode skill. ultracode-skill הוא כישור Codex לזרימות עבודה דינמיות של קידוד, שמסתגל אוטומטית לסוג המשימה בתוך מסגרת הסוכן. אם אתה בונה זרימות עבודה מבוססות Codex, הכישור הזה מוסיף תזמור גמיש ללא הגדרה מותאמת לכל פרויקט.

מספר 7. echocut. echocut הוא כלי שורת פקודה מקומי שממיר חומר וידאו גולמי לסרטון מוכן לפלטפורמה בפקודה אחת, עם FFmpeg, WhisperX ו-Ollama. אם אתה מייצר תוכן וידאו ורוצה פתרון מקומי שלם ללא ענן, כלי זה מחבר תמלול, עריכה והחלטות LLM בצינור אחד.

מספר 8. paperjury. paperjury הוא כישור Claude Code שמדמה ביקורת עמיתים לעבודות מחקר, עם שלבי סקירה, פסיקה, עדכון ואימות. אם אתה כותב מאמרים אקדמיים, הכישור הזה יכול לזהות חולשות לפני ההגשה ולחסוך דחיות מיותרות.

מספר 9. Codex Workflows. Codex-Workflows היא ספריית TypeScript לתזמור זרימות עבודה כקוד עבור סוכני Codex, עם אינטגרציית MCP. אם אתה בונה צינורות מרובי סוכנים, הספרייה הזו מאפשרת הגדרה תכנותית ומובנית שקלה יותר לתחזוקה מפתרונות מבוססי הגדרות.

מספר 10. DeepFake AI RealTime. DeepFake-AI-RealTime היא ערכת כלים C# לזיהוי ועיבוד deepfake בזמן אמת, הכוללת ניתוח LLM לווידאו, שמע ותמונות. אם אתה עובד בתחומי אבטחת מידע, ניהול תוכן או מחקר מדיה סינתטית, זהו כלי מקיף לזיהוי ואנליזה.
