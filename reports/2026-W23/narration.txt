השבוע יש לנו 10 רפוזיטוריות שרלוונטיות במיוחד לתחומים שלך.

מספר 1. lineage skill. lineage-skill הוא כלי שמזקק קורסי וידאו, PDF ותמלילים לתוך סקיל Agent ניתן להתקנה שמשמר מסגרות חשיבה, דוגמאות ונהלים עם מקורות ברורים. אם אתה לומד קורסים מוקלטים ורוצה להכניס את הידע הזה לתוך Claude Code או Codex, הכלי הזה הופך את הלמידה לנכס שניתן לשלוף בכל פרויקט.

מספר 2. memory os. Memory OS הוא מערכת הפעלה לזיכרון בת שבע שכבות ל-Hermes Agent, שמוסיפה זיכרון עקבי עם Qdrant ומזריקה הקשר רלוונטי בצורה כירורגית לכל שיחה. הכלי רץ לוקלית ועובד עם כל ספק LLM, כך שה-Agent שלך זוכר פרויקטים, החלטות ועקרונות עבודה לאורך זמן.

מספר 3. munder difflin. Munder Difflin הוא סביבת עבודה מקומית עם ממשק גרפי לניהול כמה Agent של Claude Code במקביל, תוך תיאום תקשורת וזיכרון ביניהם דרך מנצח GOD. אם אתה רוצה להריץ צוות אגנטים לפרויקטי Claude Code, הכלי הזה נותן לך ויזואליזציה ותיאום מוכנים מהקופסה.

מספר 4. skills. LLMQuant Skills הוא אוסף של סקילים מוכנים לשימוש עבור ניתוח פיננסי, מחוברים לנתוני LLMQuant ותואמים ל-Claude Code, Cursor ו-Codex. אם אתה עובד עם נתונים פיננסיים או בונה כלי AI בתחום הפיננסים, הסקילים האלה חוסכים לך בניית יכולות דומיין מאפס.

מספר 5. cc fleet. cc-fleet הוא כלי Go שמאפשר לך להוסיף מודלים חיצוניים כמו DeepSeek, Qwen ו-Kimi כ-teammates אמיתיים ב-Claude Code, תוך ניצול ארכיטקטורת ה-multi-agent של הפלטפורמה. הכלי פותח אפשרויות לחיסכון בעלויות ולשימוש במודלים מתמחים למשימות ספציפיות בתוך אותה סשן.

מספר 6. ultracode skill. ultracode-skill הוא סקיל קוד-ראשון ל-Codex שמספק זרימות עבודה מובנות לפיתוח תוכנה, תואם גם ל-Claude Code ו-Antigravity. הכלי מאפשר לנרמל את האופן שבו ה-Agent שלך ניגש לבעיות קוד מורכבות עם מתודולוגיה עקבית לאורך סשנים.

מספר 7. echocut. echocut הוא CLI לעריכת וידאו מקומית שמשלב FFmpeg עם WhisperX ו-Ollama לתמלול, זיהוי קטעים מרכזיים וייצוא גרסאות מותאמות לפלטפורמות שונות בפקודה אחת. אם אתה מייצר תוכן וידאו, הכלי הזה מבצע עריכה מבוססת AI לחלוטין לוקלית, ללא העלאה לענן.

מספר 8. paperjury. PaperJury הוא סקיל ל-Claude Code שמדמה תהליך ביקורת עמיתים לפני הגשת מאמר מחקרי, כולל ביקורת, פסיקה, הצעות לתיקון ווידוא שהתיקונים אכן פתרו את הבעיות. אם אתה כותב מאמרים אקדמיים, הכלי יכול לזהות חולשות מבניות לפני שמבקרים אמיתיים יעשו זאת.

מספר 9. Codex Workflows. Codex-Workflows הוא ספריית TypeScript שמגדירה זרימות עבודה של מספר Agent ב-Codex בקוד, עם מנגנון דוראבילי ולוח מחוונים חי בטרמינל למעקב אחר סטטוס. אם אתה בונה אוטומציות מורכבות עם Codex, הכלי נותן לך אורקסטרציה מוגדרת בקוד עם ניראות מלאה.

מספר 10. DeepFake AI RealTime. DeepFake-AI-RealTime הוא ערכת כלים מבוססת LLM ב-C# לזיהוי וניתוח מדיה סינתטית (דיפ-פייקים), המיועדת לחוקרים, עיתונאים ומבקרי תוכן שצריכים לנתח מקוריות של קבצי מדיה. אם אתה עוסק באותנטיות מדיה או ניהול תוכן, הכלי מספק יכולות זיהוי ודיווח עם גישה אתית.
