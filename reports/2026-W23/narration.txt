מספר 2. agentpet. AgentPet is a native macOS menu bar app that surfaces a desktop pet and monitors your running AI coding agents like Claude Code and Codex in real time. It keeps agent status visible during busy workdays or live sessions without requiring you to switch to a terminal.

מספר 3. ian xiaohei scenes. Ian Xiaohei Scenes is a Codex Skill that instructs AI agents to generate branded Chinese-content article illustrations by pairing a character with real objects and physical scenes. The visual metaphor approach and structured skill format are directly relevant to L&D content design and making abstract ideas concrete.

מספר 4. sandboxes. Sandboxed is a self-hosted open-source engine that spins up isolated dev sandboxes with preview URLs using a single Docker command and no Kubernetes. It is a practical foundation for building AI-assisted products where each user needs their own safe, isolated workspace.

מספר 5. agent. PentesterFlow is a human-in-the-loop CLI agent that walks penetration testers through recon, enumeration, evidence collection, and reporting while keeping the analyst in control at each step. The structured human-approval design pattern is relevant context for building safe, auditable AI workflows in regulated settings.

מספר 6. vigils. Vigils is a local-first Rust and Tauri control plane that monitors AI agent activity in real time, lets you approve sensitive actions before they execute, and keeps secrets out of agent outputs. For AI work in regulated environments like banking, its audit-log and PII-protection design directly addresses governance requirements.

מספר 7. good question. Good Question is a portable agent skill for Claude Code and Codex that sharpens research questions using scientific-method principles before any research or writing begins. For teaching, L&D design, and content work, a sharper upstream question changes the quality of everything that follows.

מספר 8. female portrait director. Female Portrait Director is a modular Codex Skill that routes AI image generation through style files and locks parameters to produce consistent portrait outputs at scale. The approach of building structured, version-controlled prompt frameworks rather than ad-hoc prompts is a transferable pattern for any content pipeline.

מספר 9. ShardBrowser. ShardBrowser is a free open-source anti-detect browser launcher with engine-level fingerprint spoofing across 170 plus device profiles, built for web scraping and multi-account workflows at scale. For AI agents that need to gather web data reliably, it addresses the bot-detection problem at the infrastructure level.

מספר 10. SoulX Transcriber. SoulX-Transcriber is an end-to-end Python framework that jointly transcribes who spoke, when, and what was said across multiple speakers in a single pass. For capturing workshop or meeting content into Obsidian notes or L&D documentation, it replaces manual transcription work.
