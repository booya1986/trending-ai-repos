Your trending A.I. repos digest for 2026, week 23.
Here are the 10 most relevant new repositories this week.

Number 1. memory-os is a 7-layer, local-first memory operating system for LLM agents that uses Qdrant for persistent vector storage and surgical context injection. For Avi, it is a direct path to an AI assistant that remembers everything across sessions, from Poalim project decisions to L&D research notes.

Number 2. munder-difflin is a TypeScript-based local multi-agent harness for coordinating multiple AI agents privately on your own machine. For Avi, it opens the door to building private, composable agent teams for L&D content and Poalim workflows without cloud dependencies.

Number 3. LLMQuant/skills is a plug-and-play library of reusable agent skills for Claude Code, Cursor, Codex, and other platforms, grounded in quantitative finance data. For Avi, it demonstrates the skills-as-code pattern that can be adapted for L&D, Obsidian, and Poalim workflows right away.

Number 4. cc-fleet is a Go tool that plugs DeepSeek, Qwen, Kimi, and other vendor LLMs directly into Claude Code as native subagents. For Avi, it means smarter cost routing across Poalim document processing and L&D content pipelines without leaving the Claude Code environment.

Number 5. ultracode-skill is a Codex-first plugin for dynamic, context-aware coding workflows. For Avi, it is a compact example of how skills extend Codex beyond single prompts into full adaptive coding pipelines, useful as a teaching reference for agent-based development.

Number 6. echocut turns raw video into branded, platform-optimized clips with one command, using FFmpeg, WhisperX, and Ollama entirely offline. For Avi, it is a direct fit for processing lecture recordings and L&D course footage into polished, multi-platform assets without any cloud costs.

Number 7. Codex-Workflows is a TypeScript library for defining multi-step Codex agent pipelines as versioned, reproducible code. For Avi, it provides a structured way to encode L&D content workflows and Poalim automation tasks so they can be reliably re-run and shared with teammates.

Number 8. paperjury is a Claude Code skill that stress-tests research papers through a four-stage review, verdict, revise, and verify cycle. For Avi, it can sharpen research-backed L&D content and internal reports before they reach Poalim stakeholders or learners.

Number 9. DeepFake-AI-RealTime is an LLM-powered C# toolkit for ethical detection, analysis, and responsible generation of synthetic media. For Avi, it is a relevant reference for building AI literacy and responsible AI content in L&D programs, especially for audiences at financial institutions like Poalim.

Number 10. LLMInternSkill is a Codex skill that automates resume tailoring, JD matching, and interview prep for LLM-focused internship searches. For Avi, it offers a ready-made model to adapt for AI career coaching and upskilling content in L&D programs.
