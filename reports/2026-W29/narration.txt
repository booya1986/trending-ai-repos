שבוע 29, 2026.
השבוע יש לנו 10 רפוזיטוריות שרלוונטיות במיוחד לתחומים שלך.

מספר 1. ditto. Ditto סורק את לוגי הסשנים המקומיים של Claude Code ו-Codex ובונה קובץ you.md שמתעד את דפוסי העבודה, ההעדפות וההחלטות החוזרות שלך. התוצאה היא זיכרון סוכן מובנה שכל עוזר AI לקוד יכול לטעון כדי להתנהג מהודעה הראשונה כ'העוזר שלך', ללא שיתוף נתונים עם הענן.

מספר 2. fable method. The Fable Method מזקק את לולאת התכנון-וביצוע-ואימות שהפכה את Claude Fable 5 לאפקטיבי לסט מיומנויות Claude Code שכל מודל יכול לאמץ. כל מיומנות עוקבת אחר מחזור Think / Act / Prove, ומונעת יישומים לא גמורים או מבוססי הזיות.

מספר 3. fable5 mode. fable5-mode מוסיף מיומנות Claude Code ומערכת hooks שמטילים את משמעת התכנון המוקפד של Fable 5 על כל מודל Claude. Opus 4.8 ומודלים קטנים יותר יתכננו, יאמתו את עצמם ויניעו סוכני משנה לפי אותן כללים שפולו Fable 5, ללא צורך במנוי Fable 5.

מספר 4. vox director. Vox Director היא מיומנות Claude Code שלוקחת נושא, מוצאת חזותיים בסגנון קולאז' נייר, כותבת תסריט, מייצרת קריינות TTS ומרכיבה סרטון הסבר מלוטש באמצעות ffmpeg, הכל בפרומפט בודד. היא מחברת ישירות את תחום יצירת תוכן הוראתי של Avi עם אוטומציה מבוססת סוכן.

מספר 5. pilotfish. Pilotfish יושב בין Claude Code ל-Anthropic API ומנתב משימות לדרגי מודל שונים: המודל החזק ביותר מטפל בתכנון והחלטות קריטיות, בעוד מודלים זולים יותר מבצעים שלבי ביצוע שגרתיים. שלב אימות מופעל לאחר כל משימת משנה כדי לתפוס רגרסיות.

מספר 6. claude antigravity agents. מיומנות Claude Code זו מוסיפה תבניות סוכן-משנה שמאצילות משימות ספציפיות (סקירת קוד, ניתוח, מחקר) ל-Google Antigravity CLI במקום לשימוש ב-Claude tokens. כל תבנית מפעילה סוכן Antigravity, מקבלת את תוצאתו ומשלבת אותה בהקשר של Claude כפלט כלי.

מספר 7. personal model. Personal Model הוא כלי macOS מקומי שאוסף אותות מהפעילות הדיגיטלית היומיומית שלך (לוח שנה, הערות, גלישה, שימוש באפליקציות) ומסנתז אותם לקובץ HUMAN.md שכל עוזר AI יכול לקרוא. הוא חושף את הפרופיל דרך MCP כדי שה'הקשר עליך' יעקוב אחריך בין אפליקציות.

מספר 8. Danus. Danus הוא מערכת רב-סוכנים שבה סוכנים מתמחים בהיגיון מתמטי משתפים פעולה תוך שיתוף זיכרון fact-graph שעוקב אחר שלבים מוכחים, עובדות נגזרות ותת-מטרות פתוחות. ארכיטקטורת הזיכרון המובנה של Danus ישירות רלוונטית לעיצוב מערכות RAG ו-knowledge-management.

מספר 9. clodex ide. Clodex IDE הוא עורך קוד מבוסס Electron, מקומי-ראשון, שנבנה במיוחד לפיתוח תוכנה אגנטי. תכונת המפתח של ה-IDE היא אפס-אמון: כל שינוי שסוכן AI מציע מוצג כ-diff לפני שנוגע במערכת הקבצים, ומאמת מבצע בדיקת נכונות לפני החלת ה-diff.

מספר 10. FableCut. FableCut הוא עורך וידאו מבוסס דפדפן ללא תלויות שרת. הציר הזמני שלו מוגדר על ידי מסמך JSON, וסוכני AI יכולים לשלוט בו דרך שרת MCP מובנה או REST API, מה שמאפשר לסוכנים לחבר, לחתוך ולרנדר וידאו מבלי לעזוב את הקשר הכלי שלהם.
